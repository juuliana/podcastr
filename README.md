## Technologies

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org)
- [TypeScript](https://www.typescriptlang.org/)

## Clone project

```bash
$ git clone https://github.com/juuliana/podcastr.git
```

```bash
# Install the dependencies
$ yarn install

# Run the JSON server
$ yarn server

# Run the project
$ yarn dev
```
